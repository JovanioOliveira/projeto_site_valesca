# Homenagem — Valesca Oliveira

Site estático em HTML/CSS/JS com slides (Swiper) para homenagem de aniversário a Valesca Oliveira.

## Descrição
Pequeno site estático com três slides que exibem imagens de fundo, texto e retratos usando a biblioteca Swiper (CDN). Projeto pensado para deploy rápido em Netlify ou qualquer servidor estático.
